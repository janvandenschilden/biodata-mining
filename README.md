# Installation

```
git clone https://github.com/janvandenschilden/biodata-mining.git
cd biodata-mining
conda install pipenv
pipenv install
pipenv run bash postBuild
```

# Run jupyter lab

```
pipenv run jupyter lab
open index.ipynb
```
